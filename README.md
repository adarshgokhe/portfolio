# Adarsh Gokhe - Personal Portfolio

A sleek, modern, and highly interactive personal portfolio website showcasing my skills, experience, and top projects in Artificial Intelligence, Web Development, and Automation.

## 🚀 Live Demo
Visit the live portfolio here: **[https://adarshgokhe.github.io/portfolio/](https://adarshgokhe.github.io/portfolio/)**

## ✨ Features
- **Dark Mode Aesthetic**: A premium dark theme with glowing neon blue and purple glassmorphism effects.
- **Dynamic Typing Animation**: An eye-catching automated typing effect in the hero section.
- **Infinite Scroll Marquee**: A continuous, smooth-scrolling banner highlighting core tech stack skills.
- **Scroll Animations**: Sections gracefully fade in as you scroll down the page.
- **AI-Generated Mockups**: Custom, professional UI mockups representing featured projects.
- **Fully Responsive**: Optimized to look great on desktops, tablets, and mobile devices.

## 🛠️ Built With
- **HTML5**: Semantic structure and layout.
- **CSS3**: Custom Vanilla CSS with Variables, Flexbox/Grid, and Keyframe Animations.
- **JavaScript (ES6)**: Intersection Observers for scroll animations and custom logic for typing effects.
- **GitHub Pages**: Fast, reliable hosting and deployment.

## 📂 Featured Projects Showcased
- **Nova AI Ultimate**: Advanced Python desktop assistant with Gemini API integration.
- **College Management System**: Robust full-stack management web platform.
- **Food Ordering App**: Interactive UI/UX mobile application prototype.
- **AI Project Backend**: High-performance Node.js API server for AI platforms.

## 📥 Local Setup
If you want to view the code locally on your machine:
1. Clone the repository:
   ```bash
   git clone https://github.com/adarshgokhe/portfolio.git
   ```
2. Navigate to the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in any modern web browser. No server is required!
